Feature

컴퓨팅 서비스

클라우드 상의 가상 머신, 가상 네트워크, 가상 스토리지 등의 서비스를 제공합니다. 학생들은 수업 및 개인 학습 용도로 활용할 수 있습니다.

  • 바로가기

상세 기능 및 서비스

JCloud는 2019년 현재 Openstack Rocky 버전을 이용해 운영 중입니다. 현재 작동 중인 서비스는 다음과 같습니다
  Nova, Cinder, Neutron, Glance, Keystone, Horizon

학과 구성원에게 제공 중인 서비스는 다음과 같습니다.
  학습 및 연구를 위한 가상 컴퓨팅 환경 제공 (머신, 네트워크, 스토리지 등)
  연구실, 동아리 및 각종 홈페이지 호스팅 (워드프레스 이미지 제공, Cinder 볼륨 기반의 백업 서비스)

기타 사용에 관한 문의는 jcloud@chonbuk.ac.kr 로 메일 주십시오.

Web IDE 서비스

웹 상에서 프로그램 개발 및 테스트를 수행할 수 있는 Web IDE 서비스를 제공합니다. C,C++,Java,Python,.Net,PHP, Node.js 등 다양한 언어를 지원합니다.

상세 기능 및 서비스

JCloud 를 기반으로 하는 Web IDE 서비스를 제공합니다. Eclipse Che를 기반으로 한 Codenvy Docker 컨테이너로 운영 중입니다. 개별 container로 구성된 workspace를 제공하여, 독립적인 개발환경을 웹 기반으로 제공하며 Local IDE 와도 연동 가능합니다. 아래 지원하는 언어와 환경을 컨테이너로 구성해주어, 쉽게 개발환경을 구축할 수 있습니다.

  •   지원 언어: C, C++, Java, Python, HTML, Node.js, .Net, Go, Kotlin, Ceylon, Ruby
  •   지원 환경: CentOS, Ubunbu, Debian, MySQL, SQLlite, Android, Maven, Apache, Tomcat, Hadoop, Rails
